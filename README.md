# Superstore Sales Analysis 🛒

## About
Analysed Superstore Sales dataset to extract business insights
using Python, Pandas, Matplotlib and Seaborn.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Key Insights
- Technology category has highest profit
- Discounts above 40% cause negative profit
- West region is most profitable
- Sales peak in Q4 (October - December)

## Files
- Superstore_Sales_Analysis.ipynb → Main analysis notebook
- superstore.csv → Dataset
